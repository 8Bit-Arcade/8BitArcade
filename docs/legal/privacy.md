# Privacy Policy

**Last Updated:** [Date TBD]

## Introduction

8-Bit Arcade ("we," "our," "Platform") respects your privacy. This Privacy Policy explains what data we collect, how we use it, and your rights.

## Summary (TL;DR)

**What we collect:**
- Wallet address (public blockchain data)
- Gameplay data (scores, timestamps)
- Basic analytics (page views, device type)

**What we DON'T collect:**
- Private keys (never!)
- Personal identity information
- Email unless you provide it
- Financial account info

**Your data is:**
- Minimally collected
- Never sold
- Blockchain data is public
- You control your wallet

## 1. Information We Collect

### 1.1 Blockchain Data (Public)

**Automatically collected:**
- Wallet address
- Transaction hashes
- Timestamps
- Token balances
- Score submissions

**Nature:** This data is public on the blockchain. We don't collect it—it's inherently available to anyone.

### 1.2 Gameplay Data

**Collected when you play:**
- Game scores
- Leaderboard rankings
- Tournament entries
- Play frequency
- Game preferences

**Purpose:** Operate leaderboards, distribute rewards, prevent cheating.

### 1.3 Technical Data

**Automatically collected:**
- IP address (temporary, for security)
- Browser type and version
- Device type (desktop/mobile)
- Operating system
- Referral source
- Page views

**Purpose:** Analytics, security, fraud prevention, optimization.

**Note:** We use privacy-respecting analytics (no Google Analytics).

### 1.4 Voluntary Data

**Only if you provide:**
- Email address (if you contact support)
- Discord username (if you join server)
- Social media handles (if you share)
- Feedback and suggestions

**Purpose:** Support, communication, community engagement.

## 2. How We Use Your Information

**We use collected data to:**
- ✅ Operate the Platform
- ✅ Distribute rewards
- ✅ Prevent cheating and fraud
- ✅ Provide customer support
- ✅ Improve user experience
- ✅ Analyze usage patterns
- ✅ Comply with legal obligations

**We DO NOT:**
- ❌ Sell your data
- ❌ Rent your data

- ❌ Use for unrelated purposes

## 5. Data Security

**Security Measures:**
- HTTPS encryption
- Secure server infrastructure
- Regular security audits
- Access controls
- No storage of private keys

**Limitations:**
- No system is 100% secure
- Blockchain data is public (by design)
- You are responsible for wallet security

**Your Responsibility:**
- Protect your seed phrase
- Use strong passwords
- Enable 2FA on wallet
- Don't share credentials

## 6. Data Retention

**How Long We Keep Data:**

| Data Type | Retention Period |
|-----------|------------------|
| Blockchain data | Permanent (public blockchain) |
| Gameplay history | Indefinitely (for statistics) |
| IP addresses | 30 days (security logs) |
| Support emails | 2 years |
| Analytics data | 1 year |

**Deletion:**
- Blockchain data cannot be deleted (immutable)
- Other data can be deleted on request (see Your Rights)

## 7. Your Rights

Depending on jurisdiction, you may have rights to:

### Access
- Request copy of your data
- Review what we've collected
- Email: privacy@8bitarcade.games

### Rectification
- Correct inaccurate data
- Update information

## 11. Blockchain Specific Privacy

**Important Considerations:**

**Pseudonymous, Not Anonymous:**
- Wallet addresses are pseudonyms
- Can potentially be linked to identity
- Blockchain analysis tools exist

**Public by Design:**
- All transactions visible
- Cannot be hidden or deleted
- Permanent record

**Enhancing Privacy:**
- Use separate wallet for gaming
- Don't reuse addresses across platforms
- Don't link to social media if you want privacy

**Username Feature:**
- Usernames are optional
- Linked to wallet address publicly
- Don't use real name if you want privacy
