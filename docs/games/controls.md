# Game Controls

## Master the Controls for Better Scores

Understanding and mastering game controls is essential for competitive play and climbing leaderboards. This guide covers all control schemes and optimization tips.

## Universal Controls

### Keyboard (Recommended for Desktop)

**All games support keyboard controls:**
- More precise than mouse
- Faster response time
- Less hand fatigue
- Preferred by top players

### Mouse/Touchpad

**Limited support:**
- Some games allow click controls
- Less optimal for competitive play
- OK for casual gaming
- Not recommended for leaderboards

### Touch (Mobile)

**Mobile-optimized controls:**
- On-screen buttons
- Swipe gestures
- Tap controls
- Varies by game

## Control Optimization

### Hardware Recommendations

**Keyboard:**
- Mechanical keyboards preferred (faster response)
- Low-latency connection (wired > wireless)
- Cherry MX Red/Silver for gaming
- Clean keys (sticky keys = death)

**Mouse (if used):**
- Gaming mouse (higher DPI)
- Wired for lowest latency
- Smooth mouse pad
- Adjust sensitivity in settings

**Monitor:**
- 60Hz+ refresh rate
- Low input lag
- Good response time (< 5ms)

### Browser Optimization

**For best performance:**
- Close other tabs (reduces lag)
- Disable browser extensions temporarily
- Use Chrome or Brave (best performance)
- Full screen mode (F11)
- Disable browser zoom (100% size)

### System Settings

**Windows:**
- Disable sticky keys
- Turn off key repeat delay
- Disable Windows key (prevents accidental menu)

**Mac:**
- Disable key repeat delays
- Turn off keyboard shortcuts
- Use wired keyboard if possible

## Practice Tips

### Control Mastery

**To get better:**

1. **Practice Mode First**
   - Learn controls without pressure
   - No gas fees
   - Focus on muscle memory
   - Experiment with techniques

2. **Focused Practice**
   - One game at a time
   - 15-30 minute sessions
   - Take breaks (prevents fatigue)
   - Track improvement

3. **Speed Drills**
   - Time your inputs
   - Practice rapid key presses
   - Improve reaction time
   - Build muscle memory

4. **Consistency Training**
   - Same time of day
   - Same hardware
   - Same posture/setup
   - Minimize variables

### Hand Positioning

**Optimal positions:**

**Chomper / Brick Breaker:**
- Fingers on arrow keys or WASD
- Thumb on space bar
- Relaxed wrist

**Space Rocks:**
- Left hand: Arrows or WASD

- Or two-handed arrows (thumb on space)

**Block Drop:**
- Left hand: Left/right arrows
- Right hand: Space bar for hard drop
- Alternating between rotation and movement

**Pixel Snake:**

## Control Lag & Latency

### Minimizing Input Lag

**Causes of lag:**
- Wireless keyboard/mouse (3-10ms delay)
- Browser overhead
- System performance
- Network issues (less relevant for controls)

**Solutions:**
- Use wired peripherals
- Close background apps
- Update browser
- Disable browser animations
- Use gaming mode (Windows)

**Testing input lag:**
- If controls feel sluggish
- Test on different browser
- Try different keyboard
- Check system performance

### When to Contact Support

**If controls:**
- Don't respond at all
- Randomly stop working
- Have severe delay (>100ms)
- Work inconsistently

**Troubleshooting first:**
1. Refresh page
2. Clear browser cache
3. Try different browser
4. Test keyboard elsewhere
5. Then contact support

## Control FAQs

**Q: Can I use a gamepad?**
A: Not yet. Gamepad support planned for Phase 3 (Months 7-12).

**Q: Can I customize key bindings?**
A: Not yet. Custom controls coming in future update. Join Discord to vote on priority.

**Q: Do I need a gaming keyboard?**
A: No, but it helps. Any keyboard works, gaming keyboards have lower latency.

**Q: Touch screen vs keyboard - which is better?**
A: Keyboard for competitive play. Touchscreen OK for casual.

**Q: My controls feel laggy, why?**
A: Try wired keyboard, close other tabs, update browser, or try Chrome/Brave.

**Q: Can I play with one hand?**
A: Some games yes (Flappy Pixel, Chomper), others harder (Block Drop, Space Rocks).

**Q: Do top players use special equipment?**
A: Most use standard gaming keyboards. Skill > equipment, but quality helps.

**Q: Controls stopped working mid-game, what happened?**
A: Browser lost focus. Click back on game window. Or page needs refresh.
